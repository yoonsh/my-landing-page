# 영어학원 랜딩페이지 제작 기획서

## 1. 프로젝트 개요

### 목적
영어학원의 강점, 수업 프로그램, 학습 성과, 상담 신청 동선을 명확히 전달하는 홍보용 랜딩페이지를 제작한다.  
방문자가 학원의 신뢰도를 빠르게 이해하고, 상담 신청 또는 문의로 자연스럽게 전환되도록 구성한다.

### 제작 환경
- Framework: Next.js
- Runtime: Node.js
- Styling: Tailwind CSS 권장
- Deployment: Vercel
- Target AI Model: Gemini Flash
- Output: 반응형 웹 랜딩페이지

---

## 2. 핵심 타겟

### 주요 대상
- 초등 / 중등 / 고등 영어학원을 찾는 학부모
- 내신, 수능, 수행평가, 영어 말하기 실력 향상을 원하는 학생
- 체계적인 관리와 학습 피드백을 원하는 학부모

### 사용자의 주요 니즈
- 우리 아이에게 맞는 수업인지 빠르게 확인
- 학원의 차별점과 성과 확인
- 수업 방식, 커리큘럼, 레벨 테스트 정보 확인
- 상담 신청을 쉽게 진행

---

## 3. 랜딩페이지 핵심 메시지

### 메인 카피 예시
> 성적 향상과 영어 자신감을 함께 만드는 영어학원

### 서브 카피 예시
> 수준별 진단, 맞춤형 커리큘럼, 꼼꼼한 학습 관리로 학생의 영어 실력을 체계적으로 성장시킵니다.

### 강조 키워드
- 수준별 맞춤 수업
- 내신 / 수능 / 회화 통합 관리
- 정기 피드백
- 소수 정예 수업
- 학부모 상담 및 학습 리포트
- 실력 향상 중심 커리큘럼

---

## 4. 페이지 구성

## Section 1. Hero

### 목적
첫 화면에서 학원의 핵심 가치와 상담 신청 버튼을 명확히 보여준다.

### 포함 요소
- 강력한 헤드라인
- 짧은 설명 문구
- CTA 버튼 2개
  - 상담 신청하기
  - 수업 프로그램 보기
- 학원 분위기 또는 학생 학습 이미지를 연상시키는 비주얼

### 예시 문구
**영어 실력, 이제 체계적으로 성장시킬 시간입니다.**  
학생의 현재 수준을 정확히 진단하고, 목표에 맞는 맞춤형 영어 학습 로드맵을 제공합니다.

---

## Section 2. 학원 강점

### 목적
방문자가 학원을 선택해야 하는 이유를 빠르게 이해하도록 한다.

### 카드형 구성 예시
1. **수준별 맞춤 커리큘럼**  
   학생의 현재 실력과 목표에 따라 학습 경로를 설계합니다.

2. **소수 정예 밀착 관리**  
   학생별 약점을 파악하고 수업 후 피드백을 제공합니다.

3. **내신과 실전 영어 동시 대비**  
   학교 시험, 수능형 독해, 문법, 어휘, 말하기까지 균형 있게 학습합니다.

4. **정기 학습 리포트 제공**  
   학부모가 학습 현황을 쉽게 확인할 수 있도록 정기 피드백을 제공합니다.

---

## Section 3. 수업 프로그램

### 목적
학부모가 자녀에게 맞는 프로그램을 쉽게 찾도록 한다.

### 프로그램 예시
- 초등 영어 기초반
- 중등 내신 집중반
- 고등 수능 영어반
- 문법 / 독해 집중반
- 영어 말하기 / 발표 수업
- 방학 특강

### UI 제안
프로그램 카드를 2열 또는 3열 그리드로 배치한다.

---

## Section 4. 학습 프로세스

### 목적
학원 등록 후 어떻게 학습이 진행되는지 신뢰감 있게 설명한다.

### 단계
1. 레벨 테스트 및 상담
2. 학생별 학습 목표 설정
3. 맞춤형 수업 진행
4. 과제 및 오답 관리
5. 정기 피드백 및 성과 점검

---

## Section 5. 성과 및 신뢰 요소

### 목적
학원의 신뢰도를 강화한다.

### 포함 가능 요소
- 성적 향상 사례
- 학생 / 학부모 후기
- 합격 또는 내신 향상 사례
- 강사진 소개
- 운영 경력
- 수업 만족도

### 예시 문구
> “문법을 어려워하던 학생이 3개월 만에 학교 시험에서 자신감을 찾았습니다.”

## Section 6. 상담 신청 CTA

- 관심 프로그램
- 문의 내용
- 개인정보 동의 체크박스

### CTA 문구
- 무료 레벨 테스트 신청하기
- 1:1 상담 예약하기
- 지금 문의하기

## 6. 기술 요구사항

### 필수
- Next.js App Router 사용

- Tailwind CSS 사용

- 이후 Google Form, Notion, Supabase, Firebase 등과 연동 가능

## 8. Gemini Flash 작업 지시문

아래 내용을 Gemini Flash에게 입력하여 랜딩페이지 초안을 생성한다.

```md
You are an expert frontend engineer and product designer.

Build a modern, responsive landing page for an English academy using Next.js, TypeScript, and Tailwind CSS. The site will be deployed on Vercel.

The landing page should target parents and students looking for a trustworthy English academy. The design should feel clean, modern, academic, and parent-friendly.

Use the following sections:

1. Hero section
- Headline: "영어 실력, 이제 체계적으로 성장시킬 시간입니다."
- Subheadline: "수준별 진단, 맞춤형 커리큘럼, 꼼꼼한 학습 관리로 학생의 영어 실력을 체계적으로 성장시킵니다."
- CTA buttons: "무료 레벨 테스트 신청하기", "수업 프로그램 보기"

2. Strengths section
- 수준별 맞춤 커리큘럼
- 소수 정예 밀착 관리
- 내신과 실전 영어 동시 대비
- 정기 학습 리포트 제공

3. Programs section
- 초등 영어 기초반
- 중등 내신 집중반
- 고등 수능 영어반
- 문법 / 독해 집중반
- 영어 말하기 / 발표 수업
- 방학 특강

4. Learning process section
- 레벨 테스트 및 상담
- 학생별 학습 목표 설정
- 맞춤형 수업 진행
- 과제 및 오답 관리
- 정기 피드백 및 성과 점검

5. Testimonials section
- Include 2–3 realistic parent/student testimonials in Korean.

6. Contact form section
- Fields: name, phone number, student grade, interested program, message
- CTA button: "상담 신청하기"

7. Footer
- Academy name, address, phone number, opening hours

Technical requirements:
- Use Next.js App Router.
- Use TypeScript.
- Use Tailwind CSS.
- Make the page fully responsive.
- Use semantic HTML.
- Add basic SEO metadata.
- Use smooth scrolling for CTA buttons.
- Keep the code clean and component-based.
- Do not use external paid assets.
- Use placeholder data where needed.
```

## 9. 1차 구현 후 검토 체크리스트

- 모바일에서 보기 좋은가?
- 글자 크기와 여백이 적절한가?
- CTA 버튼이 눈에 잘 띄는가?
- 학원 브랜드 이미지와 어울리는가?

### 기술
- `npm run dev` 실행 가능 여부

- Vercel 배포 오류 여부
- SEO title/description 설정 여부
